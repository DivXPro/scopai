# PRODUCT_SENSE.md

## 产品定位

`analyze-cli` 的产品价值在于把社交媒体内容分析拆成可重复执行的 CLI 工作流，而不是一次性的 prompt 对话。

## 核心用户

- 需要批量分析社交媒体内容的 AI agent
- 通过命令行处理数据和任务的开发者
- 需要导出统计结论的内部分析流程

## 核心体验

- 导入数据简单
- 任务创建明确
- 结果可查询、可导出
- 文档和 agent 入口清晰

## 关键取舍

- 优先 CLI 一致性，而不是 UI 体验
- 优先 DuckDB 单机链路，而不是分布式复杂性
- 优先结构化结果，而不是纯自然语言输出
