# 社交媒体分析 CLI 产品规格

## 产品目标

提供一个可被 AI agent 或人工命令行直接调用的社交媒体内容分析工具，支持：

- 导入帖子与评论数据
- 创建分析任务
- 调用大模型完成情感、意图、风险等分析
- 查询、聚合和导出结果

## 核心对象

- 平台
- 帖子
- 评论
- 模板
- 任务
- 结果

## 核心流程

1. 导入原始数据
2. 核对数据是否入库
3. 选择或调整模板
4. 创建任务并绑定目标
5. 执行分析
6. 查询和导出结果

## 当前产品边界

- 当前产品形态是 CLI，不包含前端界面
- 当前分析主路径是评论
- 结果以终端展示和导出文件为主
