# MCP Apps 扩展实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 scopai MCP Server 从 `Server` 重构为 `McpServer`，引入 `@modelcontextprotocol/ext-apps`，使 `get_post` 工具支持通过 HTML UI 在客户端 iframe 中渲染帖子信息（含图片轮播），并恢复 `platform_post_id` 查询能力。

**Architecture:** 服务端使用 `McpServer` + `registerAppTool`/`registerAppResource` 注册工具和 UI 资源；UI 为纯 HTML + CSS + JS 单文件，通过 `ext-apps` 的 `App` 类与 Host 通信；轮播为纯客户端行为，不触发额外服务端请求。

**Tech Stack:** TypeScript, `@modelcontextprotocol/sdk`, `@modelcontextprotocol/ext-apps`, Zod, Fastify

      function renderPost(post) {
        const card = document.getElementById('post-card');
        card.innerHTML = '';

        const header = document.createElement('div');
        header.className = 'post-header';

        const title = document.createElement('div');
        title.className = 'post-title';

          span.textContent = text;
          meta.appendChild(span);

          card.appendChild(buildCarousel(media));

          const noMedia = document.createElement('div');
          noMedia.className = 'no-media';
          noMedia.textContent = '暂无媒体文件';

        content.className = 'post-content';
        content.textContent = post.content || '';
        card.appendChild(content);

        if (post.tags && Array.isArray(post.tags) && post.tags.length > 0) {
          const tagsDiv = document.createElement('div');
          tagsDiv.className = 'post-tags';
          post.tags.forEach((t) => {
            const tag = document.createElement('span');
            tag.className = 'tag';
            tag.textContent = typeof t === 'string' ? t : t.name;
            tagsDiv.appendChild(tag);
          });
          card.appendChild(tagsDiv);
        }

      function buildCarousel(media) {
        const carousel = document.createElement('div');
        carousel.className = 'carousel';
        carousel.id = 'carousel';

        media.forEach((m, i) => {
          const img = document.createElement('img');
          img.src = m.src || m.url || '';
          img.alt = m.description || 'image';
          if (i === 0) img.classList.add('active');
          carousel.appendChild(img);

        nav.className = 'carousel-nav';

          images.forEach((img, i) => img.classList.toggle('active', i === index));
          counter.textContent = `${index + 1} / ${total}`;

          current = (current - 1 + total) % total;
          show(current);
        });

        nextBtn.addEventListener('click', () => {
          current = (current + 1) % total;
          show(current);
        });

        return carousel;
      }

  ```typescript
  server.registerTool('get_task_results', {
    description: 'Get analysis results for a completed task',
    inputSchema: z.object({
      task_id: z.string().describe('Task ID (required)'),
      strategy_id: z.string().optional().describe('Strategy ID (auto-detected from task steps if omitted)'),
      limit: z.number().optional().describe('Max results (default: 100)'),
      offset: z.number().optional().describe('Offset for pagination (default: 0)'),
    }),
  }, async (args) => {
    const limit = args.limit !== undefined ? `&limit=${args.limit}` : '';
    const offset = args.offset !== undefined ? `&offset=${args.offset}` : '';

    if (!args.strategy_id) {
      const task = await apiGet<any>(`/tasks/${args.task_id}`);
      const steps = task.phases?.steps ?? task.steps ?? [];
      const ids = steps
        .map((s: any) => s.strategyId ?? s.strategy_id)
        .filter(Boolean);
      if (ids.length === 0) {
        return makeTextResult({ error: 'No strategy steps found. Use strategy_id parameter.' });
      }
      const results: Record<string, unknown> = {};
      for (const sid of ids) {
        const r = await apiGet(`/tasks/${args.task_id}/results?strategy_id=${sid}${limit}${offset}`);
        results[sid] = r;
      }
      return makeTextResult(results);
    }

    const result = await apiGet(`/tasks/${args.task_id}/results?strategy_id=${args.strategy_id}${limit}${offset}`);
    return makeTextResult(result);
  });
  ```

    inputSchema: z.object({
      id: z.string().optional().describe('Internal post ID'),

      platform: z.string().optional().describe('Platform ID (required when using platform_post_id)'),

    );

  });
